# Code of Conduct

Be respectful, inclusive, and constructive. This project follows a simple, community-friendly code of conduct:

- Be kind and professional in all communications.
- Treat others with respect regardless of background.
- Report security issues privately via the repository's security contact.
- Do not post abusive, harassing, or discriminatory content.

If you see a violation, open an issue (use a private channel for sensitive reports) or email the project maintainer.

Thank you for helping keep this community welcoming.
